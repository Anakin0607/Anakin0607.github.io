# 在powershell中使用linux命令行
## chocolaty
chocolaty是windows下的一款包管理软件
[官网](https://chocolatey.org/install)

## 运行linux命令
[一些常用的工具包](https://github.com/farhanmustar/windows-config)
```
choco install `
postman `
free-download-manager `
psutils `
wezterm `
rclone `
vcxsrv `
filezilla dbeaver 7zip `
advanced-ip-scanner wireshark `
libreoffice notepadplusplus `
paint.net inkscape `
vlc `
git `
flameshot `
powertoys `
cpu-z hwmonitor wiztree `
-y
```

[psutils](https://community.chocolatey.org/packages?page=1&prerelease=False&moderatorQueue=False&tags=psutils)
